# AIRPORT.OS — Международный Терминал

### Описание
Веб-система для автоматизации аэропорта на .NET 8.

### Роли
- **Администратор:** Финансы, безопасность, логи.
- **Диспетчер:** Полосы, гейты, статусы рейсов.
- **Пассажир:** Покупка билетов, мили, онлайн-регистрация.

### Технологии
- C# (ASP.NET Core MVC)
- SignalR (Живое табло)
- Entity Framework Core (SQLite)
- CSS Grid Bento Design
